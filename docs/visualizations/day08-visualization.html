3D Minimum Spanning Tree Construction

Day 8 involves constructing a Minimum Spanning Tree (MST) in 3D space using Union-Find (Disjoint Set Union).

Key Concepts:

  • Parse 3D points from input
  • Calculate distances between all point pairs
  • Sort edges by distance
  • Use Union-Find to build MST without creating cycles
  • Sum edge weights in the final MST
