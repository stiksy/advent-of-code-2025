⚛️ Day 11: Reactor

Graph Path Counting with DFS and Memoization

Day 11 involves counting all possible paths through a directed graph using depth-first search with memoization.

Key Concepts:

  • Build directed graph from node connections
  • Use DFS to explore all paths from start to end
  • Apply memoization to avoid recomputing subproblems
  • Count paths without revisiting nodes in same path
  • Handle large path counts efficiently
